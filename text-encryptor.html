💡 将本工具部署到公网后，把地址填在这里。「智能伪装」模式会自动把原文加密到链接参数中，并把链接嵌在一段正常聊天文案里。在微信/抖音/小红书里发送时，URL 部分会自动变成可点击链接。
💡 在这里填写你的微信二维码图片链接。智能伪装会生成「点击文字 → 打开二维码图片」的文案，用户点击后可直接长按识别加微信。不要在聊天里直接发二维码图片，容易封号。

获取二维码链接的方式：
1. 把二维码图片发给微信「文件传输助手」，长按图片 → 复制图片链接
2. 把图片上传到 GitHub 仓库（见下方上传按钮）
3. 使用任何图床（如 imgbb、sm.ms）获取图片直链
💡 只在当前浏览器保存，不上传到服务器。填写后点击下方「上传二维码」按钮，图片会自动上传到你的 GitHub 仓库并获取图片直链。
